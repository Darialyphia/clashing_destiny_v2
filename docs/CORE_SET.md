# CORE SET

## LEGEND

- M: Might rune
- K: Knowledge rune
- F: Focus rune
- R: Resonance rune
- ->: exhaust
- (X): mana cost
- ((X)): destiny cost
- ~: slow speed
- ~~: fast speed
- ```: flash speed
