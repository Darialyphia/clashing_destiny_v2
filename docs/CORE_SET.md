# CORE SET

## LEGEND

- ->: exhaust
- (X): mana cost
- ((X)): destiny cost
- slow: slow speed
- fast: fast speed
- burst: burst speed
- [ ] : not implemented
- [x] : implemented
- < xxx > : tags
