# CORE SET

## LEGEND

- M: Might rune
- K: Knowledge rune
- F: Focus rune
- R: Resonance rune
- ->: exhaust
- (X): mana cost
- ((X)): destiny cost
- (slow): slow speed
- (fast): fast speed
- (burst): burst speed
- [ ] : not implemented
- [x] : implemented
